Lang Java Spring Boot V2

Apr 11, 2018 2 min read

JVM bind with IPv4

Disable IPv6 address lookups when -Djava.net.preferIPv4Stack=true

-Djava.net.preferIPv4Stack=true

Spring Boot Actuator

<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-actuator</artifactId>
</dependency>

Actuator comes with most endpoints disabled. Thus, the only two available by default are /health and /info.

management.endpoints.web.exposure.include=*

by default, all Actuator endpoints are now placed under the /actuator path
